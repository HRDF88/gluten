# Dataset

Le dataset de G.L.U.T.E.N est constitué d’un corpus documentaire structuré autour de la pizza.

## Types de sources

- Livres
- Ebooks
- Formations en ligne

## Axes couverts

- Science
- Métier
- Folklore
- Réel

## Langues

Le corpus est multilingue :

- Français
- Italien
- Anglais
- Espagnol

## Métadonnées suivies

Chaque source peut être suivie selon plusieurs critères :

- type
- titre
- auteur
- édition
- langue
- nombre de pages
- ISBN
- axe
- statut
- OCR
- PDF
- nettoyage
- regroupement thématique

## Finalité

Ce dataset sert de socle au moteur documentaire et au futur moteur de décision.
